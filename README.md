## 安装依赖
    npm install
    或 
    安装淘宝镜像,已安装跳过此步: 
    npm install -g cnpm --registry=https://registry.npm.taobao.org
    然后
    cnpm install
## 开发调试
    npm run serve
## 边调试边编译
    npm run dev
## 打包
    npm run build
## copy bunldejs 到android 项目
    npm run copy:android
## copy bunldejs 到ios 项目
    npm run copy:ios
## 同时copy bunldejs 到ios + android 项目
    npm run copy
